===
Название: top
Описание: Отображает список процессов в реальном времени с загрузкой CPU и памяти
Синтаксис: top [опции]
Пример: top -o %CPU
Примечание: Можно сортировать по CPU или памяти, использовать горячие клавиши для фильтрации

===
Название: nvidia-smi
Описание: Утилита NVIDIA для мониторинга GPU, памяти видеокарты и запущенных процессов CUDA
Синтаксис: nvidia-smi [опции]
Пример: nvidia-smi --query-gpu=utilization.gpu,memory.used --format=csv
Примечание: Отличный инструмент для отслеживания загруженности GPU

===
Название: glances
Описание: Интерактивный мониторинг CPU, RAM, дисков, сети и процессов
Синтаксис: glances [опции]
Пример: glances -w
Примечание: В режиме -w запускается веб-интерфейс

===
Название: sar
Описание: Сбор и анализ статистики системы (CPU, память, сеть)
Синтаксис: sar [интервал] [количество]
Пример: sar -u 1 5
Примечание: Утилита из пакета sysstat, выводит усреднённые данные

===
Название: vmstat
Описание: Показывает сводку по CPU, памяти, swap и процессам
Синтаксис: vmstat [интервал] [количество]
Пример: vmstat 2 5
Примечание: Полезно для диагностики «узких мест»

===
Название: ps
Описание: Отображает список процессов в системе
Синтаксис: ps [опции]
Пример: ps -eo pid,ppid,cmd,%cpu,%mem --sort=-%cpu | head
Примечание: Удобно для скриптов и выборочной фильтрации

===
Название: watch
Описание: Периодический запуск команды с обновлением в консоли
Синтаксис: watch [опции] команда
Пример: watch -n 2 nvidia-smi
Примечание: Удобно для мониторинга изменений в режиме реального времени

===
Название: kill
Описание: Завершает процессы по PID, отправляя им сигналы
Синтаксис: kill [опции] PID
Пример: kill -9 12345
Примечание: Сигнал -9 (SIGKILL) завершает процесс принудительно, но стоит использовать осторожно

===
Название: lsof
Описание: Показывает список открытых файлов и процессов, которые их используют
Синтаксис: lsof [опции]
Пример: lsof -u root
Примечание: Полезно для поиска процессов, занимающих порты или блокирующих файлы

===
Название: uptime
Описание: Отображает время работы системы, количество пользователей и среднюю нагрузку (load average)
Синтаксис: uptime
Пример: uptime
Примечание: Полезно для быстрой оценки общей нагрузки системы
